# Yashika Agrawal — Portfolio

🎬 Video Editor & Motion Designer · A&M Production · Jharkhand, India

## Features
- 3 categories: Video Edits, Reel Edits, Event Edits
- Modal video player (no redirects)
- Scroll animations & skill bars
- Mobile responsive
- Cinematic dark theme

## Stack
- React + Vite
- Pure CSS (no UI library)

## Deploy
```
npm install
npm run dev     # local dev
npm run build   # production build
```

Deploy to Vercel: connect repo, framework = Vite, done ✅
